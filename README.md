# LA Utils

A Lost Ark companion web application.

## Getting Started

### 1. Install dependencies

```bash
bun install
```

### 2. Set up environment variables

```bash
cp .env.example .env
```

### 3. Run migrations

```bash
bun run migrate
```

### 4. Start the dev server

```bash
bun run dev
```
